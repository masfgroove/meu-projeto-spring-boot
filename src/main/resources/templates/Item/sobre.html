Sobre o Projeto

Este é um projeto de aplicação web desenvolvido com Spring Boot e Thymeleaf, utilizando um banco de dados MySQL. O objetivo principal é criar uma interface para gerenciar itens, onde você pode cadastrar, editar e excluir registros.

Funcionalidades

  • Home: Página inicial que dá boas-vindas aos usuários e oferece uma visão geral do sistema.
  • Formulário: Permite o cadastro de novos itens no sistema. O formulário coleta informações como nome e descrição do item.
  • Tabela: Exibe uma lista de itens cadastrados. Oferece opções para editar ou deletar itens existentes.
  • Sobre: Página que fornece informações sobre o projeto, incluindo a descrição da implementação e tecnologias usadas.

Tecnologias Utilizadas

  • Spring Boot: Framework principal utilizado para criar a aplicação web.
  • Thymeleaf: Motor de templates para renderização de HTML.
  • MySQL: Sistema de gerenciamento de banco de dados relacional utilizado para armazenar os dados da aplicação.
  • Bootstrap: Framework CSS para estilização e responsividade da interface.

Detalhes da Implementação

O projeto foi estruturado com o seguinte layout:

  • Controladores: Responsáveis por gerenciar as requisições HTTP e retornar as páginas correspondentes.
  • Repositórios: Interfaces que estendem CrudRepository para acesso ao banco de dados MySQL.
  • Modelos: Entidades que representam as tabelas no banco de dados.
  • Templates Thymeleaf: Arquivos HTML localizados em src/main/resources/templates, usados para renderizar a interface do usuário.
